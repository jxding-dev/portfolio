# JIHYUN Portfolio 2026

Web content design portfolio built as a single-page HTML site.

## Overview

This project presents portfolio work with a desktop-style interface and visual content sections. It uses a standalone `index.html` file with local image assets stored in `web-content-design-jpg`.

## Files

- `index.html` - main portfolio page
- `web-content-design-jpg/` - portfolio image assets

## How to View

Open `index.html` directly in a web browser.

```text
D:\practice\portfolio\index.html
```
